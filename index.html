The Basics

User info

    [ui]
    username = John Tan <john@johntan.com>

*

  • tell hg who you are

  • used to "sign" your commits

put

    [ui]
    username = John Tan <john@johntan.com>

in:

  • on Unix,

    ~/.hgrc

  • on Windows,

    %USERPROFILE%\.hgrc

  • with TortoiseHg,

    Global Settings -> Commit -> Username

Change your editor also?

In .hgrc:

[ui]
editor=/my/amazing/editor

Your first repository


$ hg init vcs-demo

hg init creates a fresh, new repository

Other ways of creating a repository


# in the current directory
$ hg init
# in the folder called `mynewproject`
$ hg init mynewproject

Term: Repository

  • "repo" in short

Stores data

  • history

  • files

  • files at each state

Check your repo


$ hg status
? index.html

hg status shows you...

  • ...status of files in your local repo

Some common status codes:


? = not tracked
A = added
C = clean
M = modified

How do you tell hg to track your file?

$ hg add index.html
$ hg status
A index.html

But hg add is only a statement of intent

Really do it


$ hg commit -m "first step towards world domination"

hg commit...

saves the current state

creates a checkpoint in history
